## Installation
แน่นอนว่าจะใช้งาน **Ansible** ได้จะต้องติดตั้ง **Python** ก่อน หลังจากนั้นก็ติดตั้ง **Ansible** ผ่าน **pip** ซึ่งเป็น packakge manager ด้วยคำสั่ง
```bash
$ pip install ansible
```
